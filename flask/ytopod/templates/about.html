About

Ytopod is a webapp for creating podcast feeds from youtube videos. It will download given youtube video and convert it to audio format. Then it adds the audio to podcast rss file and presents it to podcast players to fetch. It requires one user, which will be created after initial setup. Currently it generates only one feed accessible at location /download/feed.xml. This endpoint is also secured by username and password, so the podcast player needs to support http auth for podcasts.

The app can be run using docker and directions are in the github repo. Currently it supports only one feed, saves metadata into sqlite database saved in the docker container and allows to delete the podcast videos. There are more features that I have in mind and some of them should be present in the readme file in the mentioned github repo.
